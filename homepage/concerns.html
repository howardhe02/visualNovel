Concerns

The one big concern when doing this project is that time may limit what we are able to do. Since the game is a dating sim and is meant to give the player as much freedom as possible, many outcomes must be taken into consideration. The time provided to complete this project is scarce though, so many ideas may be cut, meaning the holes created when the functions are cut must be patched quickly. For example, a player name input was cut, so we must replace every player name with something else.
